- Octo now supports image input! For multimodal models, try dragging and
  dropping images into your terminal
- Added @-autocompletion for file names: type @ and a snazzy new filepicker will
  open up
- In Collaboration Mode, you can whitelist tools for Octo to auto-accept with
  the new "Always allow..." menu item in the permission prompt
- Added support for using shell commands to access API keys, for better
  password-manager support.
- Improved Vim emulation
- And a lot more! Run `octo changelog` to see the full list.
